# Rep
The idea for this tool is its like Anki(spaced repitition) but inside the Command Line Interface(CLI). It should support all the important features of Anki such as supporting .apkg files, creating decks, adding of flashcards, and syncing to AnkiWeb.

I'm thinking we use a tool called [ink](https://github.com/vadimdemedes/ink?tab=readme-ov-file) which Claude and Gemini also use for their CLI interfaces. I'm not sure if this is the best tool for the job, but it's a start.

If you have any suggestions please do tell me as I'm still visioning the full scope of this project.
